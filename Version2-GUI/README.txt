    Para compilar o programa, introduza no terminal: javac Main.java
    Para executar o programa, introduza no terminal: java Main [option]

    Para [option] pode ser de duas maneira:
        1) não fornecer ficheiro -> java Main, 
            nesta opção oprograma apresenta também uma opção de teste da árvore
            no entanto, o utilizador deve fornecer os dados
        2) fornecer um ficheiro CSV -> java Main file.csv
            nesta opção o ficheiro fornecido é CSV mas sem classificação,
            após a árvore ser criada, irá, para todas as entradas do ficheiro CSV
            classificar os dados.

    Note, é necessário ter o java instalado e JRE também.
    

    Estes programas foram criados e executados em máquina  64 bits Windows 10 
Home com 16 GB de RAM, Intel Core i7-8550U CPU @ 1.80GHz 1.99 GHz 8 cores
